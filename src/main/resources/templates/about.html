About URL Shortener

A powerful, fast, and secure URL shortening service built with modern technologies.

What is URL Shortener?

URL Shortener is a web service that takes long URLs and converts them into short, shareable links. This is particularly useful for social media, email marketing, and anywhere you need to share a link but want it to be concise and professional-looking.

Our service not only shortens URLs but also provides detailed analytics about how your links are being used, including click counts, geographic data, and referrer information.

  • Click tracking and analytics

API Documentation

Our service provides a RESTful API for developers who want to integrate URL shortening functionality into their applications.

Available Endpoints:
  • POST /api/urls/shorten - Create short URL
  • GET /api/urls/{code} - Get URL info
  • GET /{code} - Redirect to original URL
Response Format:
{
  "success": true,
  "message": "URL shortened successfully",
  "data": {
    "shortCode": "abc123",
    "shortUrl": "http://localhost:8080/abc123",
    "originalUrl": "https://example.com/very/long/url"
  }
}
